# Checklist de Materiales para Cannes

## Materiales de Submission

### Video Case Study
- [ ] Archivo MP4, H.264
- [ ] Duración: ≤ 120 segundos
- [ ] Resolución: 1920x1080 mínimo
- [ ] Subtítulos en inglés (hardcoded)
- [ ] Sin branding de agencia/cliente visible
- [ ] Música licenciada (guardar documentación)

### Imagen de Presentación
- [ ] Resolución: 1920x1080
- [ ] Formato: JPG o PNG
- [ ] Sin texto superpuesto
- [ ] Representa visualmente la campaña

### Descripción Escrita
- [ ] Descripción corta (100 palabras)
- [ ] Descripción larga (300 palabras)
- [ ] En inglés perfecto (revisar con nativo)
- [ ] Incluye insight, ejecución y resultados

---

## Documentación de Soporte

### Brief Original
- [ ] Documento de brief del cliente
- [ ] Objetivos de campaña originales
- [ ] Fechas de briefing

### Evidencia de Implementación
- [ ] Fechas exactas de campaña (inicio - fin)
- [ ] Fotos de instalación en comedores
- [ ] Lista de comedores participantes
- [ ] Contratos/acuerdos con dueños

### Resultados Cuantificables
- [ ] Datos de ventas before/after
- [ ] Número de impresiones calculado
- [ ] Encuestas a comensales (resumen)
- [ ] Earned media value

### Verificación de Cliente
- [ ] Carta de aprobación de Pulp
- [ ] Datos de contacto del CMO
- [ ] Confirmación de que pueden verificar datos

---

## Checklist Final Pre-Submit

### Contenido
- [ ] Video < 2 minutos ✓
- [ ] Subtítulos verificados por nativo ✓
- [ ] Sin agencia/cliente visible ✓
- [ ] Imagen 1920x1080 ✓
- [ ] Descripciones en inglés ✓

### Documentación
- [ ] Brief original adjunto
- [ ] Fechas de implementación claras
- [ ] Evidencia de campaña real
- [ ] Contacto CMO para verificación

### Resultados
- [ ] Números verificables documentados
- [ ] Metodología de medición clara
- [ ] Fuentes citadas

### Legal
- [ ] Música licenciada
- [ ] Permisos de imagen de personas
- [ ] Aprobación final del cliente
- [ ] No hay claims que no se puedan respaldar
